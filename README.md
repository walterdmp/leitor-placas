## Leitor de placas de automóveis

O presente projeto foi desenvolvido como parte da disciplina de computação gráfica e busca ajudar no controle de acesso dos veículos no Instituto Federal do Sul de Minas campus Machado.

O projeto utiliza uma IA para leitura das placas dos carros e verificar no banco de dados se o veículo está autorizado a entrar.
